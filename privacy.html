3. 정보주체의 권리 (Arts. 15–22 GDPR)

정보주체는 언제든 다음 권리를 행사할 수 있으며, 운영자는 합리적인 기간 내에 응답합니다.

  • 열람권 (Art. 15) — 본인에 대해 저장된 데이터 확인. 관리자 계정 보유자는 내 계정 메뉴에서 프로필 + 활동 로그를 바로 확인할 수 있습니다.
  • 정정권 (Art. 16) — 부정확한 정보의 수정 요청.
  • 삭제권 / 잊혀질 권리 (Art. 17) — 계정·로그의 삭제 요청. 감사 목적의 로그는 사용자 식별자가 [deleted:<id>] 형태로 익명화되어 보존됩니다.
  • 처리 제한권 (Art. 18) — 정보의 사용 중지 요청.
  • 데이터 이동권 (Art. 20) — 본인 데이터를 기계 판독 가능 JSON 형태로 내보내기. 관리자 UI에서 다운로드 가능.
  • 거부권 (Art. 21) — 정당한 이익 기반 처리에 대한 반대.

권리 행사 요청은 info@bpmedia.net로 보내주세요.

4. 제3자 제공 및 국외 이전 (Arts. 44–49)

BP미디어는 원칙적으로 개인정보를 제3자에게 제공하지 않습니다. 다만 서비스 운영을 위해 다음 처리 수탁자의 인프라를 이용합니다.

  • Cloudflare, Inc. (미국) — 호스팅, DNS, CDN, Workers(서버리스 실행 환경), D1 데이터베이스, R2 오브젝트 저장소. GDPR 적정성 결정 대체 수단으로 SCC(Standard Contractual Clauses)가 적용됩니다.
  • Kakao AdFit (대한민국) — 게시글 페이지 광고 서빙. 광고 식별 쿠키 관련은 Kakao의 개인정보 처리방침을 따릅니다.
  • Cloudflare Turnstile — 로그인 페이지 봇 방지 CAPTCHA. 쿠키를 사용하지 않으며 개인 식별 정보를 수집하지 않습니다.

5. 보안 대책 (Art. 32)

  • 관리자 비밀번호는 PBKDF2-SHA256 (100,000 iterations, 16 byte salt)로 해시 저장. 평문 저장 없음.
  • 세션 토큰은 HMAC-SHA256 서명, HttpOnly + Secure + SameSite=Lax 쿠키로 전달.
  • 로그인 시도는 IP + 아이디 조합으로 15분 10회 제한.
  • 전 구간 TLS (Cloudflare), HSTS 활성화.
  • 운영자 권한은 메뉴·행동별 세분화된 매트릭스로 관리되어 최소 권한 원칙을 따릅니다.
  • 감사 로그에 모든 운영자 조작이 기록되며 90일 보관 후 삭제·익명화됩니다.

6. 아동 정보

BP미디어는 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 부주의로 수집된 정보가 있다면 즉시 삭제합니다.

7. 변경 고지

본 방침이 변경될 경우 개정일과 주요 변경 내용을 이 페이지와 사이트 푸터에서 공지합니다. 중대한 변경은 운영자 페이지의 버전 기록에도 남습니다.
