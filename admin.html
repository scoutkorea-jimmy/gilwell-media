생성 시각: 새 글 작성 시 자동 기록

슬라이드 전용 이미지를 올리면 기사 하단에서만 별도 슬라이드로 노출됩니다.

에디터 관리 (A–C)

각 에디터의 내부 실명을 입력하세요. 실명은 관리자만 볼 수 있으며 기사에는 "Editor A" 형식으로 표시됩니다. 현재는 A~C만 사용합니다.

게시판 카드 간격

Korea, APR, WOSM, Scout People 게시판 카드의 상하 간격입니다. 최소 5px부터 설정할 수 있습니다.

px

게시판 D-day 설정

카테고리 배너 오른쪽 큰 영역에 행사명과 D-day를 표시합니다. 날짜를 비워두면 기존처럼 게시글 수가 표시되며, 모바일에서는 표시되지 않습니다.

글머리 태그 관리

A는 공통 태그, B는 게시판 전용 태그입니다. 태그를 추가한 뒤 드래그해서 공통과 각 게시판 영역 사이를 이동할 수 있습니다.

B · Korea 전용
B · APR 전용

일반 페이지 공유 설정

홈, 게시판, 검색, 도움 페이지를 공유할 때 노출될 제목과 설명을 설정합니다.

특히 `홈 / index` 설명은 구글과 네이버 검색 결과의 사이트 소개 문구로 활용됩니다.

평균적으로 반영 확인까지 1~10분 정도 걸릴 수 있으며, 카카오톡·페이스북·X 같은 플랫폼은 자체 캐시 때문에 더 오래 보일 수 있습니다.

푸터 좌측 전체 문구

푸터 좌측 문구는 아래 필드만 수정하면 자동으로 정리된 형태로 반영됩니다.

미리보기

권장 1200×630px, PNG/JPG, 1MB 이하. 너무 작은 이미지는 공유 미리보기가 비어 보일 수 있습니다.

각 도구에서 발급된 meta verification의 content 값만 입력하면 됩니다.

히어로 기사 설정 (최대 5개)

메인 히어로에 노출할 기사를 최대 5개까지 선택하세요. 자동 슬라이드로 순환되며, 기사 추가 목록에는 현재 공개된 최신 기사만 표시됩니다.

선택된 히어로 카드는 PC 버전과 모바일 버전의 이미지 위치·확대를 각각 따로 저장합니다.

이미지를 100% 아래로 줄이면 같은 이미지를 흐린 배경으로 깔아 빈 공간이 어색하지 않게 보강합니다.
