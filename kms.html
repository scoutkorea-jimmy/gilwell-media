API 가이드 / 연동 매뉴얼

BP미디어 백엔드 전체 엔드포인트 목록입니다. 인증이 필요한 항목은 Authorization: Bearer <token> 헤더를 포함하세요.

Base URL https://bpmedia.net DB Cloudflare D1 (SQLite)
인증 방법
  1. POST /api/admin/login 에 비밀번호를 전송해 JWT 토큰을 발급받습니다.
  2. 이후 모든 admin 전용 요청에 Authorization: Bearer <token> 헤더를 포함합니다.
  3. 토큰 유효기간은 24시간입니다. 만료 시 재로그인이 필요합니다.
