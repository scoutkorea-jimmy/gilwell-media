# Surface Library

> 이 문서는 페이지를 최상위 축으로 다루지 않고, **feature 조합 결과로 보이는 surface node**를 모아두는 허브입니다.

## 원칙

- surface는 기능/모듈보다 아래 레이어다.
- surface 문서는 “이 페이지가 어떤 기능을 조합해서 보이는가”만 설명한다.
- 같은 기능 설명을 surface 문서에 복사하지 않는다.
