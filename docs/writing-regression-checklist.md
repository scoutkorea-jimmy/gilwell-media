# Writing Regression Checklist

## 관리자 페이지

- `/admin.html` 로그인 가능
- 새 글 작성 폼 진입 가능
- 제목 필수 검증 동작
- 작성자 선택 가능
- 게시 날짜 선택 가능
- 공통/게시판별 태그 선택 가능
- 대표 이미지 업로드 가능
- 본문 이미지 5개 제한 동작
- AI 지원 여부 저장 가능
- 임시저장 및 복원 동작
- CAPTCHA 완료 후 게시 가능

## 홈페이지 모달

- `/korea.html`, `/apr.html`, `/wosm.html`에서 글쓰기 진입 가능
- 관리자 로그인 후 모달 열림
- 제목 필수 검증 동작
- 작성자 선택 가능
- 게시 날짜 선택 가능
- 공통/게시판별 태그 선택 가능
- 대표 이미지 업로드 가능
- 본문 이미지 5개 제한 동작
- AI 지원 여부 저장 가능
- 임시저장 및 복원 동작
- CAPTCHA 완료 후 게시 가능

## 결과 확인

- 작성된 글이 해당 게시판 목록에 노출
- 개별 글 페이지 `/post/:id` 정상 렌더링
- 비공개 글은 직접 접근 차단
